SASSOU DÉCOR - SITE WEB
=============================================

INSTRUCTIONS D'INSTALLATION

1. CRÉER LA STRUCTURE DE DOSSIERS
   - Créez un dossier nommé "sassou-decor"
   - Dans ce dossier, créez un sous-dossier nommé "images"

2. PLACER LES FICHIERS
   - Copiez tous les fichiers fournis (index.html, style.css, script.js, admin.js) dans le dossier principal
   - Placez votre fichier "banniere.jpg.JPG" dans le dossier principal
   - Placez toutes vos photos de produits dans le dossier "images"

3. OUVRIR LE PROJET DANS VISUAL STUDIO CODE
   - Ouvrez VS Code
   - Allez dans Fichier > Ouvrir un dossier
   - Sélectionnez le dossier "sassou-decor"
   - Installez l'extension "Live Server" (recommandé)

4. LANCER LE SITE
   - Avec Live Server : cliquez sur "Go Live" en bas à droite de VS Code
   - Sans Live Server : ouvrez directement le fichier index.html dans votre navigateur

5. UTILISATION DU SITE
   - Page principale : affiche tous les produits avec possibilité de filtrer
   - Panier : cliquez sur l'icône panier pour voir et gérer vos articles
   - Commandes : utilisez les boutons WhatsApp pour passer commande
   - Paiement : via MonCash au numéro +509 3490 2252

6. ADMINISTRATION
   - Mot de passe par défaut : "admin123"
   - Cliquez sur le cadenas en haut à droite pour accéder à l'admin
   - Vous pouvez ajouter, modifier ou supprimer des produits
   - Vous pouvez changer le mot de passe admin

7. AJOUTER VOS PRODUITS
   - Accédez à l'administration avec le mot de passe
   - Cliquez sur "Ajouter un produit"
   - Renseignez les informations et le nom du fichier image
   - Placez l'image correspondante dans le dossier "images"

8. HÉBERGEMENT GRATUIT
   - Pour héberger gratuitement, utilisez GitHub Pages :
     1. Créez un compte GitHub
     2. Créez un nouveau dépôt
     3. Téléversez tous vos fichiers
     4. Activez GitHub Pages dans les paramètres du dépôt

9. ESPACE POUR FUTUR BOUTON
   - Une zone est prévue dans la section "Contact et Informations"
   - Vous pouvez l'utiliser pour ajouter un nouveau bouton ou fonctionnalité

CONTACT TECHNIQUE
Si vous avez des questions, n'hésitez pas à contacter votre développeur.
